## Image Optimization Recommendations

### Priority Actions:
1. **Convert to AVIF/WebP format** - Reduce file sizes by 70-90%
2. **Target file sizes:**
   - Hero backgrounds: <200KB (currently 1.4-1.5MB)
   - Trust signals: <50KB (currently 1.2MB each)
   - Service images: <100KB (currently 1.1-1.3MB)
3. **Measure actual dimensions** - Add width/height to all Image components
4. **Generate responsive sizes** - Create 2-3 breakpoints for mobile/tablet/desktop

### Optimization Tools:
```bash
# Install Sharp for Next.js automatic optimization
npm install sharp

# Or use manual conversion tools:
# AVIF: npx @squoosh/cli --avif auto
# WebP: npx @squoosh/cli --webp auto
```

### Image Compression Commands:
```bash
# Compress PNG to AVIF (70-90% smaller)
npx @squoosh/cli --avif '{"cqLevel":33,"cqAlphaLevel":-1,"denoiseLevel":0,"tileColsLog2":0,"tileRowsLog2":0,"speed":6,"subsample":1,"chromaDeltaQ":false,"sharpness":0,"tune":0}' ./hero/*.png

# Compress JPEG to AVIF
npx @squoosh/cli --avif auto ./services/*.jpeg

# Generate WebP fallbacks
npx @squoosh/cli --webp '{"quality":75,"target_size":0,"target_PSNR":0,"method":4,"sns_strength":50,"filter_strength":60,"filter_sharpness":0,"filter_type":1,"partitions":0,"segments":4,"pass":1,"show_compressed":0,"preprocessing":0,"autofilter":0,"partition_limit":0,"alpha_compression":1,"alpha_filtering":1,"alpha_quality":100,"lossless":0,"exact":0,"image_hint":0,"emulate_jpeg_size":0,"thread_level":0,"low_memory":0,"near_lossless":100,"use_delta_palette":0,"use_sharp_yuv":0}' ./trust-signals/*.png
```

## SEO Best Practices Summary

### Alt Text Guidelines:
- **Length:** 125 characters maximum (search engine optimization)
- **Descriptive:** Explain what's in the image AND its purpose
- **Keywords:** Include relevant keywords naturally (not stuffing)
- **Context:** Add location-specific terms (Orange County, California, OC)
- **Accessibility:** Write for screen readers first, SEO second

### Title Attribute Guidelines:
- **Concise:** Shorter than alt text, focuses on key benefit
- **Actionable:** Suggests what user can do/learn
- **Brand:** Include "EyeCare Center" or practice name
- **Keywords:** Primary keyword + modifier

### File Naming Best Practices:
- **Lowercase only** - Prevents URL case sensitivity issues
- **Hyphens not underscores** - Google treats hyphens as word separators
- **Descriptive keywords** - Explain image content
- **Location terms** - Include city/region when relevant
- **No special characters** - Only letters, numbers, hyphens
- **Avoid dates** - Unless required for versioning

### Loading Strategy:
- **Above-fold images:** `priority={true}` + `loading="eager"`
- **Below-fold images:** `loading="lazy"`
- **Hero backgrounds:** Always priority
- **Trust badges:** Lazy load (appear after scroll)
- **Service cards:** Lazy load, responsive sizes
